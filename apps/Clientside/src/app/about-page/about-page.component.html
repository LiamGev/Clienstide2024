  • Description:

    • Ik ontwikkel een Wiki-webapplicatie voor de game Terraria, waarin gebruikers informatie kunnen vinden over items en vijanden. De applicatie biedt inzicht in welke vijanden specifieke items droppen en onder welke omstandigheden. De applicatie richt zich op fans van de game die snel toegang willen hebben tot gedetailleerde informatie over de interactie tussen items en vijanden in het spel.

  • Functional Requirements:
    • ID Functional Requirement Acceptatiecriteria Prioriteit (MoSCoW)
    FR1 Als een gebruiker wil ik een account kunnen aanmaken, zodanig dat ik toegang krijg tot alle functionaliteiten. Gebruiker kan succesvol registreren met een uniek e-mailadres en wachtwoord. Must Have
    FR2 Als een gebruiker wil ik kunnen inloggen, zodanig dat ik mijn account kan beheren. Gebruiker kan inloggen met correcte inloggegevens en krijgt toegang tot hun persoonlijke dashboard. Must Have
    FR3 Als een gebruiker wil ik artikelen kunnen toevoegen, zodanig dat ik informatie kan delen over de game. Gebruiker kan een nieuw artikel aanmaken, opslaan, en wijzigen via een formulier. Must Have
    FR4 Als een gebruiker wil ik een overzicht van alle artikelen, zodanig dat ik makkelijk informatie kan vinden. Gebruiker ziet een lijst met artikelen, gesorteerd op datum of populariteit. Must Have
    FR5 Als een gebruiker wil ik artikelen kunnen bewerken, zodanig dat ik mijn bijdrage kan aanpassen. Alleen de auteur van een artikel kan dit bewerken, mits ingelogd. Must Have
    FR6 Als een gebruiker wil ik mijn favoriete artikelen kunnen opslaan, zodanig dat ik snel toegang heb. Gebruiker kan een artikel markeren als favoriet en dit terugvinden in een persoonlijke lijst. Could Have
    FR7 Als gebruiker wil ik een overzicht zien van alle items, zodanig dat ik details over specifieke items kan bekijken. - De lijst toont minimaal itemnaam, type, en zeldzaamheid. - Ik kan klikken op een item om meer details te zien, zoals beschrijving en gebruik. Must Have
    FR8 Als gebruiker wil ik een overzicht zien van alle vijanden, zodanig dat ik details over specifieke vijanden kan bekijken. - De lijst toont minimaal vijandnaam, type, en sterkte. - Ik kan klikken op een vijand om meer details te zien, zoals hun sterkte en unieke kenmerken. Must Have
    FR8 Als gebruiker wil ik kunnen zien welke vijanden een bepaald item droppen, zodanig dat ik gericht kan zoeken naar specifieke drops in het spel. - Op de detailpagina van een item zie ik een lijst met vijanden die het droppen. - De lijst toont dropkans en eventuele condities. Must Have
